# Gulp 4.0 Static Starter Kit

## Getting Started

```
$ npm i -g gulp-cli
$ npm i
$ gulp
```

## Default Tasks

- Pug Compile
- Sass Compile
- CSS Autoprefix
- JavaScript ES6 Compile
- Image Minify
- BrowserSync(auto reload)

## Testing Environment

```
$ gulp -v
CLI version 2.1.0
Local version 4.0.0

$ node -v
v10.15.3
```
